Movement Is Not Admission

The Missing Production Control in a Tokenized Money-Market-Fund Collateral Sandbox

Darwin Rampai · Version 1.0 · 16 July 2026
Read the determination Version 1.0 release Machine-readable evidence

Determination

The July 2026 GDF and ISDA tokenized-money-market-fund sandbox demonstrated collateral movement under assumed eligibility. It did not test whether a live tokenized fund satisfied the production admission boundary.

Native FINOS CDM: 5 PASS / 3 PARTIAL / 9 FAIL. CDM plus TFII profile: 17 / 17 PASS. Admission control: PASS, FAIL and UNKNOWN branches executed. Five public records: 5 UNKNOWN / 0 admission decisions manufactured.

Boundary

UNKNOWN is not a finding that an instrument is unsafe or ineligible. The five public records did not contain the receiver-specific ownership, control, reconciliation, valuation, haircut, custody and default-access evidence required to make a production decision.

Production Inspection

A production inspection requires one named instrument, one pledgor-and-receiver workflow, the actual admission schedule and authorized operating records. The controlled output is an integrity receipt, eligibility determination or evidence hold, Unknown register and calculation hashes. The first inspection is USD 3,000 prepaid.
