# Tokenized Financial Instrument Integrity: Public Evidence Package

This package accompanies Darwin Rampai's paper *The Token Is Not the Financial
Instrument* (Version 1.0, 16 July 2026).

## Supported Result

Five United States primary filings describe five materially different
relationships among token movement, authoritative ownership records,
transfer-effectiveness boundaries, reconciliation, and correction. Similar
tokenized-share forms therefore cannot be normalized as one ownership or
transfer model without preserving source-native authority and timing.

## Public Contents

- the final paper in PDF, DOCX, HTML, and Markdown;
- a bounded five-record machine sample;
- a reduced claim-to-evidence matrix;
- one deterministic transfer-boundary classification example;
- two black-and-white evidence figures;
- the public record contract, rights basis, correction policy, and bounded
  downstream-use analysis;
- a permanent correction channel and public correction register; and
- a hash-bound release manifest.

## Reproduction

1. Read the bounded public sample.
2. Apply the rules in the classification example to each represented
   `transfer_effective_event`.
3. Confirm that the five records resolve into four observed timing groups plus
   one explicit Unknown.
4. Compare the result to the public claim-to-evidence matrix and the paper's
   architecture table.
5. Verify every packaged file against `SHA256SUMS.txt`.

The package proves the bounded classification result. It does not reproduce the
complete maintained registry or every normalized source locator.

## Controlled Boundary

The complete maintained registry, normalized source receipts, calculation
receipts, correction feed, monitoring implementation, complete source locators,
and customer delivery package are excluded. Their commercial use requires
written terms.

## Limitations

The sample is purposive and limited to five United States filings. It is not a
legal opinion, valuation source, regulated benchmark, audit, investment
recommendation, or allegation of wrongdoing. Private operating records are not
available. Unknowns remain Unknown.
